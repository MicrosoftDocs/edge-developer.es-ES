# JsValueRef typedef
Una referencia a un valor de JavaScript.  
  
## Sintaxis  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## Observaciones  
 Un valor de JavaScript es uno de los siguientes tipos de valores: undefined, null, Boolean, String, Number u Object.  
  
## Requisitos  
 **Encabezado:** jsrt. h
