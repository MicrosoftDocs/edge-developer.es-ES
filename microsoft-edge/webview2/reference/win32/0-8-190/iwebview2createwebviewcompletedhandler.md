# interfaz IWebView2CreateWebViewCompletedHandler 

> [!NOTE]
> Esta interfaz puede modificarse o no estar disponible para las versiones posteriores a la versión de SDK 0.8.355. Consulta la [referencia](../../../webview2-api-reference.md) de la API más reciente.

```
interface IWebView2CreateWebViewCompletedHandler
  : public IUnknown
```

La persona que llama implementa esta interfaz para recibir la vista en WebView creada a través de CreateWebView.

## Resumen

 Miembros                        | Descripciones
--------------------------------|---------------------------------------------
[Invoke](#invoke) | Se llama para proporcionar al implementador el estado de finalización y el resultado de la llamada de método asincrónica correspondiente.
